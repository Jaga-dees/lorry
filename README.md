# 🚚 New Indian Cargo Carriers - React Portfolio

A modern, responsive portfolio website for New Indian Cargo Carriers, a lorry booking agency with 30 years of experience in logistics and transport across India.

## 📋 Features

- ✅ Professional & modern design
- ✅ Fully responsive (mobile, tablet, desktop)
- ✅ 8 main sections with smooth scrolling
- ✅ Company information & services showcase
- ✅ Fleet showcase with vehicle details
- ✅ Client testimonials
- ✅ Contact information
- ✅ SEO optimized

## 🎨 Sections Included

1. **Header** - Navigation menu with sticky positioning
2. **Hero** - Eye-catching banner highlighting 30 years of service
3. **About** - Company history and key highlights
4. **Services** - 6 transport service offerings
5. **Fleet** - Modern vehicle fleet showcase
6. **Why Choose Us** - 6 compelling reasons to choose NICC
7. **Testimonials** - Client feedback and reviews
8. **Contact** - Full contact details and business info
9. **Footer** - Quick links and company info

## 🚀 Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Jaga-dees/lorry.git
   cd lorry
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open in browser**
   - The app will automatically open at `http://localhost:3000`

### Build for Production
```bash
npm run build
```
